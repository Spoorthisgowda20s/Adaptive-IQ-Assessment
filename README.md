 Adaptive IQ Assessment

An interactive web application designed to evaluate users through an adaptive IQ assessment system. The platform dynamically presents questions, records user responses, and securely stores user information and assessment data using MongoDB Atlas.

 Features
 
 User Registration and Login Authentication
 Adaptive IQ Assessment with Multiple Questions
 Stores User Responses and Assessment Data
 Maintains User Login Details Securely
 MongoDB Atlas Integration for Cloud Database Storage
 User-Friendly and Responsive Interface
 Real-time Question Navigation and Submission
